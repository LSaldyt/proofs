\problem{6.2.8}

Let $a,b,c$ be integers.
\begin{enumerate}
    \item If $a|b$ and $b|a$, then $a = +/-b$.
    \item If $a|b$ and $b|c$, then $a|c$.
\end{enumerate}

\solution

\part 

If $a|b$ and $b|a$, then $b = ax$ and $a = by$, so $a = axy$, which is only true when $xy$ is one.
So either $x = -1$ and $y = -1$ or $x = 1$ and $y = 1$. Substitute these into the original statements, and either $a = -b$ or $a = b$.

\part

If $a|b$ and $b|c$, then $b = ax$ and $c = by$. So $c = axy$, and thus $c = az$ for $z = xy$, so $a$ divides $c$.
