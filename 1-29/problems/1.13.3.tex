\problem{Contrapositives}

Find the contrapositives of the following statements. Write in positive terms where possible.

\begin{enumerate}
    \item If $x < 0$, then $x^2 > 0$.
    \item If $x \neq 0$, then there exists $y$ for which $xy = 1$.
    \item If $x$ is an even integer, then $x^2$ is an even integer.
    \item If $x + y$ is odd and $y + z$ is odd, then $x + z$ is odd.
    \item If $f$ is a polynomial of odd degree, then $f$ has at least one real root.
\end{enumerate}

\solution

\part 

\begin{enumerate}
    \item If $x^2 \leq 0$, then $x \geq 0$.
    \item If for all $y$, $xy \neq 1$, then $x = 0$.
    \item If $x^2$ is an odd integer, then $x$ is an odd integer.
    \item If $x + z$ is even, then $x + y$ is even and $y + z$ is even.
    \item If $f$ does not have at least one real root, then $f$ is a polynomial of even degree.
\end{enumerate}
