\problem{If-then (1.3)}

Consider the statements:
\begin{enumerate}
    \item P: All dogs eat meat
    \item Q: Rome is in Italy
    \item R: Chocolate prevents cavities
    \item S: The moon is made of green cheese
\end{enumerate}

Determine which of the following are true:
\begin{enumerate}
    \item If P, then Q
    \item If P, then R
    \item If R, then S
    \item If S, then Q.
    \item If Q, then S.
\end{enumerate}

\solution

\part 

Assume P and Q are true, but R and S are false.

\begin{enumerate}
    \item If P, then Q: True (Vacuously)
    \item If P, then R: False
    \item If R, then S: True (Vacuously)
    \item If S, then Q: True (Vacuously)
    \item If Q, then S: False
\end{enumerate}
